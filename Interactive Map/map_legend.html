MAP LEGEND

The map is colour coded based on the amount of funding each country received from China. The countries were placed in categories for easier classification

Category 1 painted in gainsboro (hex code #DCDCDC) represents countries than did not receive any funding from China since 2005.

Category 2 painted in aliceblue (hex code #F0F8FF) represents countries than received less than US $10 million in net investments from China since 2005.

Category 3 painted in lightskyblue (hex code #87CEFA) represents countries than received less than US $100 million in net investments from China since 2005.

Category 4 painted in deepskyblue (hex code #00BFFF) represents countries than received less than US $1 billion in net investments from China since 2005.

Category 5 painted in royalblue (hex code #4169E1) represents countries than received more than US $1 billion in net investments from China since 2005.

Category 6 painted in lightsalmon (hex code # #FFA07A) represents countries that have a negative net FDI inflow from China since 2005.
