China's Investments in Africa

A look at China's Foreign Direct Investments (FDI) in Each African Country Since 2005.

Background

The purpose of this website is to graphically represent China’s foreign direct investments in each country in Africa. This site is hosted via Netlify and coded using HTML, CSS and JavaScript.

How to Navigate the Interactive Map

  1. Click the hamburger navigation bar on the top left
  2. Select the pages you want to navigate to
  3. Once the interactive map page is open, hover over any country to display its name
  4. Click the country to open the bar chart, which will open in a new tab

Resources

The interactive map was created using SVG, the rough map was obtained from Wikimedia and then edited to fit the data.

Data about each country's net FDI inflows was obtained from the World Bank website.

Data about China’s FDI into each country was obtained via The Statistical Bulletin of China's Outward Foreign Direct Investment published by China's Ministry of Commerce (MOFCOM). The data was collected and presented in an excel file that was accessed via the China Africa Research Initiative (CARI) website. The charts were created using CanvasJS.
